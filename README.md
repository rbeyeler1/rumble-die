# Rumble Die

A party game where the bass decides your fate — not your skills.

## About

Rumble Die is a competitive card-collection drinking game for adults 21+. Place a six-sided die on a Bluetooth speaker, crank the music, and let the bass vibrations roll the die for you. Each roll triggers a different action:

- **Rolls 1–4**: Duel another player for a numbered card
- **Roll 5**: Total Wipe — lose all your cards back to the pool
- **Roll 6**: Group Flip-Off — everyone competes for a card

First player to hold cards 1, 2, 3, and 4 at the same time wins.

## Running Locally

Clone the repo and open the page in your browser:

```bash
git clone https://github.com/rbeyeler1/rumble-die.git
cd rumble-die
open index.html
```

Or serve it locally:

```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```
